# Big Data Cloud Analytics Dashboard

## Features
- Data Cleaning Pipeline
- Supabase Cloud Database
- Interactive Analytics Dashboard
- AI Business Insights using Gemini API
- Responsive UI

## Technologies Used
- React
- Tailwind CSS
- Node.js
- Express
- Supabase
- Gemini API
- Pandas

## Installation

### Backend

cd backend
npm install

### Frontend

cd frontend
npm install

### Main Folder
npm install
npm run dev

## AI Integration
This project uses Google Gemini API free tier to generate business insights based on food delivery analytics.
